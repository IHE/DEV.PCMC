# Approved Change Proposals

[`CP/`](../) directory.

See the [Change Proposal (CP) Workflow](https://github.com/IHE/DEV.documentation/blob/main/playbooks/cp-workflow.md)
for the full process.

> This README also keeps the otherwise-empty `CP/Approved/` directory in git.
> Leave it in place.
